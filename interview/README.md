## 注意点

* 不要太紧张
* 穿着得体,有礼貌
* 关注一下最近发生的与你所面试职位的一些技术大事或发展，比如前端就发布了es6，然后又可以balbala说一大堆了
* 一般面试是20～25分钟，只要在这区间，就很可能通过了
* 前边也说到要保持每轮面试前后一致性
* 提一些自己与众不同的特点如果问到，并举例说明，举例一定要真实。
